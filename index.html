¿Qué es HTML?

HTML es un lenguaje de marcado que por sus siglas en inglés se llama Hyper Text Markup Languaje y es el lenguaje en el que está escrito el documento que lee el navegador, ya que cuando entramos a una página web, lo que sucede es que el navegador web está pidiendo un documento donde se encuentra almacenada esa página. Entonces, si HTML es un lenguaje con el que se escribe el documento que contiene la información de una página web, el cual es leído por el navegador, ¿Cómo es que se escribe? ¿cuál es su estructura? Básicamente, una página web se puede comparar con una estantería, la cual podría ser el cuerpo de la página, la cual podría componerse en varios niveles o espacios que, a su vez, podrían tener o dividirse en varias secciones. Etiquetas HTML

Son muchísimas las etiquetas que existen para estructurar el contenido en el documento HTML, se suelen clasificar en Etiquetas semánticas, etiquetas multimedia, etiquetas de cabecera, etiquetas de formulario y etiquetas de scripting. Voy a tratar de practicar cada una de ellas con el devenir del curso y dejarlo todo registrado en esta página. Donde:

  • <"DOCTYPE html">: Informa al navegador que versión de HTML se usó.
  • <Head>: Provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.
  • <meta charset="UFT-8">: Lo utilizamos para declarar el uso de lenguas con alfabeto germánico o latino.
  • <meta http-equiv="X-UA-Compatible" content="IE=edge">: Muestra los sitios habilitados para Google Chrome Frame mediante la tecnología de renderizado de Google Chrome, lo que te permite acceder a las últimas funciones HTML5, así como a las funciones de rendimiento y seguridad de Google Chrome sin interrumpir en modo alguno el uso habitual del navegador
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Básicamente, sirve para definir qué área de pantalla está disponible al renderizar un documento, cuál es nivel de escalado que puede realizar el usuario, así como si el navegador debe mostrarla con algún zoom inicial.
  • <Title>: Esta etiqueta nos permite establecer o darle un título a la página, el cual se escribirá en la barra de título del navegador y se mostrará en las páginas de resultados de los buscadores.
  • <body>: Representa a todo el contenido del documento HTML para ser mostrado por el navegador y solo puede haber 1 elemento <body> en el documento HTML
Como vemos, cada página web comienza con la etiqueta <HTML> y dentro de ella van <head> y <body>, donde el head se escribe la información sobre el documento que no se muestra al usuario paro que es relevante para su funcionamiento; y en el body va todo el contenido que se mostrará al usuario a través del navegador. De esto también podemos darnos cuenta que dentro de una etiqueta podemos contener una o varias etiquetas con contenidos adicionales.
