¿Qué es HTML?

HTML es un lenguaje de marcado que por sus siglas en inglés se llama Hyper Text Markup Languaje y es el lenguaje en el que está escrito el documento que lee el navegador, ya que cuando entramos a una página web, lo que sucede es que el navegador web está pidiendo un documento donde se encuentra almacenada esa página.

Entonces, si HTML es un lenguaje con el que se escribe el documento que contiene la información de una página web, el cual es leído por el navegador, ¿Cómo es que se escribe? ¿cuál es su estructura?
Básicamente, una página web se puede comparar con una estantería, la cual podría ser el cuerpo de la página, la cual podría componerse en varios niveles o espacios que, a su vez, podrían tener o dividirse en varias secciones.
